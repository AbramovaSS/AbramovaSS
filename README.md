### 🔍 Ключевой QA опыт и экспертиза

*   **Виды тестирования:** Функциональное, Smoke, регрессионное.
*   **Глубокая аналитика трафика:** Активно использую **Charles Proxy** для перехвата, подмены (Breakpoints/Map Local) и валидации сетевых запросов.
*   **Работа с API:** Понимание архитектуры REST, структуры HTTP-запросов, заголовков и кодов статус-ответов.
*   **Опыт в Agile/Scrum:** Привычна к работе в условиях частых релизов, быстрых спринтов и постоянно меняющихся требований без потери качества.
